# Dotfiles

This repository contains my full NixOS system configuration, my Home-Manager setup, and all user configuration files for my desktop environment and applications.

## Installation

### Clone the repository

```bash
git clone https://github.com/EnzoSergiani/.dotfiles.git ~/.dotfiles
cd ~/.dotfiles
```

### Run the setup

```bash
chmod +x ./setup.sh
sudo ./setup.sh
```
